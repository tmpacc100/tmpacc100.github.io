👁️ Markdown プレビュー

Markdownプレビューアとは？

入力したMarkdown（マークダウン）記法を、リアルタイムでHTML形式に変換してプレビュー表示するツールです。 GitHubのREADME作成やブログ記事の下書き、メモ書きなどに便利です。

主な特徴

  • リアルタイム更新: 左側のエディタに入力すると、右側のプレビューに即座に反映されます。
  • GitHub Flavored: 一般的なMarkdown記法に加え、テーブルやコードブロックなどの拡張記法にも対応しています。
  • 安全な処理: 入力内容はどこにも送信されず、ブラウザ内だけで処理されるため、プライバシーも安心です。

よくある質問 (FAQ)

画像の埋め込みはできますか？
はい、`![alt](img-url)` の形式でWeb上の画像を表示できます。ローカル画像のアップロード機能はありません。
HTMLタグは使えますか？
はい、Markdown内にHTMLタグを記述してそのまま表示させることも可能です（サニタイズは行われませんのでご注意ください）。
